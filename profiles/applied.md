# Profile: applied

Computation + sanity-checking bundle. Optimize for *a trustworthy number* with an audit trail.

- **Primary skill:** `dimensional-analysis` leads; `mathematical-rigor` backs derivations.
- **Every equation** passes a units check; **every number** carries units, uncertainty, and
  sig-figs (default from `config.sig_figs`).
- **Two independent checks** on each result: a limiting-case/asymptotic check and an
  order-of-magnitude (Fermi) cross-check.
- Symbolic steps verified with the configured `cas` when non-trivial.
- Output ends with an **uncertainty budget** and the cross-checks performed.
